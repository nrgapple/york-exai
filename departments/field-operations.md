# Field Operations Charter

## Mission

Support the actual pest-control work happening in the field.

## Inputs

- scheduled jobs
- active check-ins
- voice memos
- inspection notes
- callback requests

## Outputs

- structured field updates
- closeout-ready service notes
- follow-up tasks
- callback classification

## Responsibilities

- keep job state current without requiring long typing
- track what was found, what was done, and what still needs handled
- flag prep issues, access issues, and repeat activity
- protect documentation quality from the field side

## Escalate When

- job cannot be completed as scheduled
- prep or access failure blocks work
- likely callback is detected
- compliance-sensitive documentation is missing
