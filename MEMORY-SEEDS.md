# MEMORY-SEEDS.md

These are durable facts future York ExAI workspace memory should absorb early.

- York ExAI supports a real pest-control business in central Pennsylvania.
- Core service area assumptions: York, Harrisburg, Lancaster, and surrounding territory.
- This is an exterminator operation, not generic field service.
- Default service scope is general pest plus wood-destroying insect workflows.
- First-class pest domains include ants, bees and wasps, termites, bed bugs, spiders, roaches, rodents, and common household pest issues.
- The business is owner-led and field-heavy.
- The system must help run route days, inspections, callbacks, closeout, invoicing, collections, compliance, and accounting handoff.
- Default field check-in channel is iMessage.
- Voice memos are a first-class field input because the owner does not want to type long updates in the field.
- Owner-facing agents should sound blunt, practical, and blue-collar central Pennsylvania without turning into parody.
- Internal agents should stay sharp, technical, analytical, and explicit.
- Compliance-sensitive changes must be approved before implementation.
- QuickBooks is optional downstream sync or export, not the required system core.
